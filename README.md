# React Weather App 🌤️

A simple and responsive Weather Application built using **React + Vite**.
This application allows users to search for a city and view its weather information such as temperature, humidity, and maximum temperature.


## 🚀 Features

* Search weather by city name
* Display temperature
* Show humidity level
* Weather icons based on conditions
* Clean and responsive UI
* Built with modern React concepts


## 🛠️ Tech Stack

* React.js
* Vite
* JavaScript
* CSS
* Material UI Icons


## 📂 Project Structure

```
src

 ├── SearchBox.jsx

 ├── InfoBox.css
 ├── SearchBox.css
 └── main.jsx
```

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/Nikhilsingh46/react-weather-app.git
```

Go to project folder

```
cd react-weather-app
```

Install dependencies

```
npm install
```

Run the project

```
npm run dev
```
